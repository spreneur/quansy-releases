# Quansy AI - Releases

Quansy AI 데스크탑 앱의 공개 릴리스 및 자동업데이트 피드를 배포하는 레포입니다.

- 앱 다운로드(.dmg)와 자동업데이트 피드(`latest.json`)가 여기 [Releases](../../releases)에 올라갑니다.
- 앱 코드는 비공개이며, 빌드 산출물만 이 레포로 공개 배포됩니다.
- 설치 후에는 앱 안에서 새 버전을 자동으로 받습니다(재설치 불필요).

> Quansy AI는 토스증권 Open API 기반의 규칙 기반 퀀트 자동매매 데스크탑 앱입니다. 투자에는 원금 손실 위험이 있으며, 모든 투자 결정과 책임은 사용자에게 있습니다.
